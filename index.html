About This Demo

This is a Firebase Authentication demonstration with the following features:

  • Login/Signup: Email and password authentication
  • Page Protection: Protected pages that require authentication
  • Auto-redirect: Automatic redirection based on authentication status
  • Password Reset: Forgot password functionality
  • Real-time Auth: Authentication state updates in real-time

Try These Actions:

  1. Try accessing the "Protected Home Page" without logging in
  2. Create a new account via "Sign Up"
  3. Login with your credentials
  4. Access the protected page after authentication
  5. Test the logout functionality
